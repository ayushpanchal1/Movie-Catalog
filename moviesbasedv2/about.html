What is Moviebased?


Moviebased is a university project.


Made by students Ayush Panchal, Piyush Pingalkar, Dhiraj Pandit and Kushalraj Pandey; the project involves learning REST API framework and testing/implementing IMDB API.



Why APIs?


We decided to learn REST APIs as APIs are the future. (or even present at this point)


Most monolithic projects now use micro services to ease up production and development, we feel that API knowledge is crucial in current day web development.



Where can I get IMDB API?


You can get the API here on rapidAPI. It contains documentation and example responses too.
